# Lerico Resolution Nodes

Simple resolution helper nodes for ComfyUI.

## Nodes

### Resolution Forge V2

`Resolution Forge V2` is the recommended node.

It includes a custom HTML/CSS/JS interface, cleaner controls, live result previews where possible, common resolution presets, aspect-ratio tools, image-based sizing, exact sizing, output caps, and a customizable resolution string output.

Use this node for new workflows.

### Resolution Forge

`Resolution Forge` is the original version.

It is kept for compatibility with older workflows, but it is deprecated. It uses the older default widget layout and has fewer features than V2.

## Notes

This project is tested primarily by the author. I am the sole tester, so bugs, edge cases, and unexpected behavior are possible.

If something behaves strangely, prefer testing with `Resolution Forge V2` first.
